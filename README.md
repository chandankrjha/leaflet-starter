# leaflet-starter

![README02](./img/README02.png)

Start Leaflet easily.  
- [Leaflet](http://leafletjs.com) 
- [webpack](https://webpack.js.org)  

<br>

## Usage

![README03](./img/README03.png)

<br>

Install package

```

npm install

```

<br>

build

```

npm run build

```

<br>

dev

```

npm run dev

```
