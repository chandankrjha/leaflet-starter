# leaflet-starter

![README02](img/README02.png)

![README05](img/README05.png)

Start Leaflet easily.  
- [Leaflet v1.5.1](http://leafletjs.com) 
- [webpack v4.39.1](https://webpack.js.org)  
- node v12.7.0
- npm v6.10.0

<br>

## Usage

![README04](img/README04.png)

<br>

Install package

```

npm install

```

<br>

build

```

npm run build

```

<br>

dev

```

npm run dev

```
